This plugin allows WordPress users to embed interactive flash health education tutorials within a wordpress page or post. 

== Description ==

This plugin allows embedding an interactive flash tutorial simply by putting the code <code>[PEI-H1N1,480,360]</code> where 480 and 360 are the width and height of the display area.

More tutorials and in more languages will be available at :

http://wp.patient-education.com


== Installation ==

1. Go to your WordPress admin area, then go to the "Plugins" area, then go to "Add New".
2. Search for "Patient-Education" (sans quotes) in the plugin search box.
3. Click "Install" on the right, then click "Install" at the top-right in the window that comes up.
4. Activate the plugin.
5. Put the code [PEI-H1N1,480,360] in your post or page.

== Frequently Asked Questions ==

= How do I use the plugin after installing it? =

Edit your page/post and put in the code [PEI-H1N1,480,360]

= Can I modify the size? =

Yes! The second parameter (480) is for width and third parameter (360) is for height. You can change these parameters but try to keep them of same proportions for best results.

= What is an X-Plain Health Education Tutorial? =

Unlike videos or printed handouts where the user may not be able to understand the presented informaiton X-Plain tutorials are fully narrated, self paced presentations with graphics and animations that help explain surgeries, medical procedures and conditions, chronic diseases, and healthy living topics.  Each tutorial also presents the user with questions and gives them feedback based on their answers.  The tutorials are self paced and very engaging.  

= Who is the intended audience for the X-Plain tutorials? =

X-Plain tutorials are a great resouce of unbiased health information for the general population, patients, health consumers, employees, students, and healthcare providers.

= What about low literacy learners? =

X-Plain was developed for the widest audience.  The text of each tutorial is written at a 6th grade readability level and all text is fully narrated.

= Can X-Plain be impelmented to document comprehension and completion and be hooked to my EMR? =

Yes, X-Plain is routinely implemented in healthcare and wellness sectors where the educational activity is documented and the data is passed to client systems.
Contact http://wp.patient-education.com/contactus for details.

= Are there any other languages? =

Yes. H1N1 tutorial is available also in Spanish, French and Arabic.
Please visit http://wp.patient-education.com for languages availability.

= Are there any other tutorials beside H1N1 ? =

Yes! Patient-Education has the largest interactive health library. We shall be releasing new tutorials for public use periodically. With the same plugin installed, you just have to use a different code. Example : [PEI-DIABETES,480,360]

= Can I customize or change the skin? =

Yes. Contact us for more details on this. 
http://wp.patient-education.com/contactus
